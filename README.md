Para executar o projeto existem duas formas:

Método I -
1. Instala o Android Studio e o SDK do Android.
2. Abre o projeto na IDE.
3. Conecta um celular ou cria um AVD.
4. Roda o projeto.

Método II -
1. Passa o arquivo "TurismoGO.apk" presente no diretório.
"app\build\outputs\apk\debug\" para um celular
2. Executa o arquivo para instalar.

Obs.: Em alguns aparelhos, o Método II exige que o usuário permita
a instalação de aplicativos de terceiros. Uma vez que o apk gerado
é de debug e não está assinado.
